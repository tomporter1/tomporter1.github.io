Valorant Companion

A comprehensive Valorant companion app designed to track statistics, manage game data, and provide insights across multiple seasons.

C# WPF Entity Framework

Project Overview

This project is a stat tracker for the popular game Valorant , developed during my time in the Sparta Global C# SDET training program. Initially created as part of a week-long challenge, I continued refining it post-training to add advanced features like tracking rank changes, supporting multi-season data storage, and enabling long-term queries (e.g., finding games with the most kills). The app offers a user-friendly interface for navigating game statistics, adding or editing agents, and visualizing historical gameplay data.

The application is built using a layered architecture, with a business layer managing interactions between the GUI and database through Entity Framework Core. This structure ensures clean separation of concerns and maintainability. Key features include multi-season data support, rank tracking, and customizable queries for deep dives into game statistics. The project demonstrates my ability to design scalable applications that evolve over time while maintaining usability.
