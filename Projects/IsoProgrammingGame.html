Learn Programming Game

Teaching basic programming through puzzles

Unity C# Game Development Prototyping

Project Overview

During my time at Gaia Technologies, this was an R&D project that was development to be used as a prototype for a full app that was developed after my placement finished.

This game was designed to teach programming concepts to children in a fun and interactive way. The game was set in an isometric world where the player would have to solve puzzles by programming the movements of the character to reach the goal.

Technical Details

Key Features

  • Simple programming language developed within unity
  • Level generator for rendering other levels based on a data file
